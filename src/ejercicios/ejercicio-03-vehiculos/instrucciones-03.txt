| Gestión de vehículos en venta |

    Debe desarrollar un sistema para la gestión de vehiculos.
    Minimo son 3 atributos para la clase Padre.

    Clase Padre
        - Vehiculo

    Clases Hijas
        - Automóvil
        - Motocicleta
        - Camión

    Métodos obligatorios
        - calcularPrecioFinal(): number
        - aplicarDescuento(): number
        - mostrarInformacion(): void

    Métodos extra sugeridos
        - calcularImpuesto(): number
    
    Cada vehículo debe calcular su precio final considerando impuestos o descuentos.
    Mostrar el inventario disponible.
